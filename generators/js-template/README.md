# sao-js-template

> my remarkable SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao js-template my-project
```

### From git

```bash
sao hjinbin/sao-js-template my-project
```
